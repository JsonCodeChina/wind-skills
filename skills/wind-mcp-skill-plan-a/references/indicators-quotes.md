# 行情类指标词典

> 何时读：入参需填行情相关指标名时（如 `indexes`），每次核对 | 权威于：Wind 行情指标名词典 | 不覆盖：自然语言字段 `question` / `query` / `metricIdsStr`

本表是 Wind 行情类指标名的**唯一权威词典**。凡入参需要填写行情类指标的调用都以本表为准——
最常见是 `*_price_indicators` 工具的 `indexes` 参数。
每次构造这类参数都回本表逐字核对；表内没有的字段不要传，不要凭记忆或猜拼写。

## 适用品种

| 类别 | 适用品种 |
|---|---|
| 元数据 / 基础行情 / 盘口五档 / 成交统计 / 流动性 / 多周期涨跌幅 / 市值 / 股息 / 涨跌停 / 盘中异动 / 资金流向 / 盘前盘后 | **股票 + 基金 + 指数 通用** |
| 估值 | **股票 + 指数为主**（基金的 PE/PB 极少有意义） |

## 字段陷阱

1. **必须照抄字面**:`涨跌`、`涨跌幅`、`5分钟涨跌幅`、`基于Wind算法的量比`,**括号 / 全角字符 / 阿拉伯数字** 一字不差。
2. **极易混淆的同名变种**:`涨跌` (单位:元) ≠ `涨跌幅` (单位:%) ≠ `涨跌BP` (单位:bp,债券专用) ≠ `5分钟涨跌幅`。
