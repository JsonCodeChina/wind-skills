# 技术指标词典

> 何时读：入参需填技术分析相关指标名时（如 `indexes`），每次核对 | 权威于：Wind 技术指标名词典 | 不覆盖：自然语言字段 `question` / `query` / `metricIdsStr`

本表是 Wind 技术类指标名的**唯一权威词典**。每次构造这类参数都回本表逐字核对；
表内没有的字段不要传，不要凭记忆或猜拼写。

## 适用品种

技术指标适用于 **股票 + 基金 + 指数 通用**。schema 列了不代表后端实现了或当前品种有数据。

---

## 字段陷阱

1. **必须照抄字面**:`指数平滑异同移动平均`、`6周期相对强弱指标`、`12周期相对强弱指标`,**括号 / 全角字符 / 阿拉伯数字** 一字不差。
2. 遇到字段返空或报错，**不要在快照工具里反复试拼写**，直接切 NL 类工具兜底：
   - 多周期涨跌幅 / 历史走势 / 资金流时间序列 → `get_stock_technicals` / `get_global_stock_technicals` / `get_index_technicals` (`{question}` 自然语言)
