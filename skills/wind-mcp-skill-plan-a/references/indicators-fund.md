# 基金指标词典

> 何时读：入参需填基金相关指标名时（如 `indexes`），每次核对 | 权威于：Wind 基金指标名词典 | 不覆盖：自然语言字段 `question` / `query` / `metricIdsStr`

本表是 Wind 基金类指标名的**唯一权威词典**。每次构造这类参数都回本表逐字核对；
表内没有的字段不要传，不要凭记忆或猜拼写。

## 适用品种

基金指标适用于 **基金 / ETF / LOF 专属**（传给股票/指数返空）。

---

## 字段陷阱

1. **必须照抄字面**:`最新净值增长率`、`近一周净值增长率`、`七日年化收益率`，**括号 / 全角字符 / 阿拉伯数字** 一字不差。
2. schema 列了不代表后端实现了或当前品种有数据。遇到字段返空或报错，直接切 NL 类工具兜底。
