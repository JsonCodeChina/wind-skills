# 东方财富结构化数据查询

## 查询能力

### 1. 可识别的查询对象
* 股票（A 股、港股、美股）
* 板块、指数、股东
* 企业发行人、债券、非上市公司
* 股票市场、基金市场、债券市场

### 2. 可返回的数据类别

可查询的结构化数据主要包括：

* **市场行情**（最新价格、涨跌表现、盘口明细等）
* **交易与量化指标**（技术统计、资金流向等）
* **财务与报表指标**（收入、利润、财务比率等）

### 3. 组合查询与**用量限制**

查询可同时包含多个实体、指标和时间范围。**受服务端能力约束，单次请求需要遵循以下配额：**

* **实体数量**：一次请求最多包含 **5 个** 股票或其他金融实体。
* **超限处理**：请求中的实体超过限额时，仅处理排列在前的 5 个，并在说明文件中记录截断情况。

> **示例：** “查询 A、B、C、D、E、F 六只股票的营收” -> 仅返回前五只。

### 4. 结果文件

每次成功执行后会生成以下文件：

- 一个保存结构化数据的 `xlsx` 工作簿
- 一个记录查询内容、结果含义和必要提示的 `txt` 说明文件

## 查询示例

### 基础指标查询

- 东方财富近三年的主要盈利指标
- 五粮液最近一年的营业收入和净利润
- 新能源汽车 ETF 的阶段业绩表现

### 实时行情与盘口查询

- 当前 300750 的实时买单
- 微软现在的最新价和涨跌幅
- 中证 500 当前点位和成交额

### 多实体、多指标、多时间查询（注意配额）

* **限额内示例**：对比上证 50、沪深 300、中证 1000（3 个实体）年初以来的涨幅（1 个指标）。
* **限额内示例**：宁德时代、比亚迪（2 个实体）最近一年的营收、毛利、净利（3 个指标）。
* **超限场景**：查询“沪深市场前十大权重股的 PE”时，系统只处理前 5 名。

------

## 查询边界

- 本 Skill 用于获取金融结构化数据和关系数据，不承担主观研判、投资建议或资讯解读。
- 查询语句必须指出明确的金融实体。
- 自然语言请求会被解析为查询对象、指标与时间区间，并整理为结构化结果文件。
- 对于能力范围内的请求，优先生成结构化数据，并附带结果说明文件供后续处理。


### 3. 安装依赖


```bash
pip3 install httpx pandas openpyxl --user
```

## 快速开始

### 在工作目录下执行

```bash
python3 {baseDir}/scripts/get_data.py --query "宁德时代最近一个月的涨跌幅"
```

### 输出示例

```
xlsx: /path/to/miaoxiang/mx_finance_data/mx_finance_data_9535fe18.xlsx
描述: /path/to/miaoxiang/mx_finance_data/mx_finance_data_9535fe18_description.txt
行数: 42
```

### 输出文件说明

| 文件 | 说明 |
| --- | --- |
| `mx_finance_data_<查询id>.xlsx` | 结构化数据表，包含请求的实体与指标 |
| `mx_finance_data_<查询id>_description.txt` | 包含查询逻辑说明、字段含义及**配额截断提示** |
